# 🦔 LogGuard — Монитор целостности логов

**Простой. Практичный. Продаваемый.**

---

## Для кого это?

| Кто | Зачем |
|-----|-------|
| 🏥 Медклиники | Доказать: истории болезней не меняли |
| 🛒 Интернет-магазины | Доказать: заказы не подделаны |
| 🏦 Маленькие банки | Соответствие требованиям регуляторов |
| 🖥️ Хостинг-провайдеры | Доказательство для клиентов |

---

## Что делает?

1. 🔍 Следит за папкой с логами
2. 🔐 Каждые 5 минут считает хеш каждого файла
3. 📊 Сохраняет доказательство в базу
4. 🚨 Если хеш изменился — шлёт уведомление

---

## Как установить?

```bash
# 1. Скопируйте папку logguard
cp -r logguard /your/project/

# 2. Настройте config.json
{
  "папка_для_логов": "/var/log/myapp/",
  "email_для_уведомлений": "security@yourcompany.com"
}

# 3. Запустите
python3 logguard.py &
Пример вывода
